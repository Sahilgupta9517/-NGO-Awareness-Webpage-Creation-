NGO Awareness Website – InAmigos Foundation

Project Overview
Developed a responsive NGO awareness website for InAmigos Foundation to showcase the organization's mission, social impact initiatives, ongoing projects, and community engagement opportunities. The website aims to increase awareness about education, environmental sustainability, digital inclusion, animal welfare, and youth empowerment programs.

Key Features
Responsive and modern user interface
Hero section highlighting NGO mission and vision
Project showcase for social initiatives (SEVA, Bachpanshala, JEEV, UDAAN, PRAKRITI, and VIKAS)
Impact statistics section displaying organizational achievements
Call-to-action sections for volunteering, partnerships, and donations
Clean navigation and user-friendly design
Mobile-friendly layout
